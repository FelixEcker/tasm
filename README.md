# t(heft)asm
Assembler for the theft fantasy-cpu.

## Usage
`tasm -i <sourcefile> [-o <outfile>] [-f <format>]`

| Short | Long     | Description                       |
| ----- | -------- | --------------------------------- |
| -i    | --in     | Specify the input to be assembled |
| -o    | --out    | Specify the output file name      |
| -f    | --format | Specify the output format         |
